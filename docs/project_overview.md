# IPL Analytics Platform

## Overview

The IPL Analytics Platform is an end-to-end data analytics project built using SQL Server, Python, Git, GitHub, and Power BI.

The project transforms historical IPL ball-by-ball cricket data into meaningful business insights through database design, data modeling, SQL analytics, and interactive visualizations.

The platform enables users to analyze player performance, team performance, venue statistics, match outcomes, and season-level trends using structured data and analytical reporting.

---

## Project Goals

The primary goals of this project are:

- Design a normalized relational database.
- Implement data ingestion and transformation processes.
- Build analytical SQL queries.
- Develop meaningful business KPIs.
- Create interactive Power BI dashboards.
- Demonstrate end-to-end data analytics skills.

---

## Problem Statement

IPL data is available in raw CSV format and contains detailed ball-by-ball information across multiple seasons.

However, raw data alone does not provide meaningful insights.

This project aims to transform the raw dataset into a structured analytics platform capable of answering business questions related to:

- Team performance
- Player performance
- Match analysis
- Venue analysis
- Season analysis

---

## Data Source

Dataset:

- IPL Ball-by-Ball Dataset (2007–2026)

Dataset Characteristics:

- Approximately 286,000+ records
- 64+ columns
- Ball-by-ball level granularity
- Multiple IPL seasons
- Match, player, team, venue, and performance data

## Database Design

The database is designed using a normalized schema consisting of:

- Seasons
- Teams
- Players
- Venues
- Matches
- Deliveries

The design supports efficient querying, reporting, and dashboard development.
